# Key-Value Memory Network

Memory 部分以key和value的形式存储不同的内容。key的部分用于寻址，value的部分作为输出。

这样的key value的具体形式可以是例如key是主语和谓语，value是宾语；也可以key是一个窗口，value是窗口的中间的词汇。

1. Key Hashing
2. Key Addressing
3. Value Reading
